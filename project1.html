HTML

Definition

HTML stands for Hyper Text MarkUp Language.It is like a skelton of a web page. It controls the structure of it. HTML is the main type of document on the web.

HTML is made up of
  • Text Content-what you see .
  • Mark Up- descibes how it looks like.
  • Refences to images and or videos.
  • Links to the other pages.

This video gives a good overview.

TAGS

HTML is made up of Tags.They come in angled brackets. They usually come in pairs called, opening tags and closing tags. Images do not have any closing tags.They are called 'void tags'.

  • Element - An element refers to everything within a set of opening and closing tags. There are Inline elements,like 'span' which have a string like content as opposed to Block elements, like 'div' which are invisible blocks with content or other elements within them too.
  • Attribute- This is a property of an HTML element.

CSS

Definition

CSS stands for Cascading Style Sheets. Cascading means most specific rule applied.This idea is known as Inheritance.The tree like structure is because of DOM-Document Object Model. CSS allows you to use specific syntax and rules to change how elements look on a page like the size of a font, color,background,position, etc.

How it works

CSS is a code written to control the style of HTML elements through Selectors. There are element selectors, id selectors and class selectors. The declaration or declaration block in CSS is the section of a style rule that defines the styles to be applied.
